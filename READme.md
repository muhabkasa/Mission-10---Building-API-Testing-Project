# Mission 10 - Building API Testing Project

Sebuah project REST API sederhana untuk manajemen data film (Movie App) yang dibangun menggunakan **Node.js**, **Express**, dan database **MySQL**.

## Fitur (CRUD)
- **GET /movies**: Menampilkan semua daftar film.
- **GET /movie/:id**: Menampilkan detail film berdasarkan ID.
- **POST /movie**: Menambahkan data film baru.
- **PATCH /movie/:id**: Memperbarui data film yang sudah ada.
- **DELETE /movie/:id**: Menghapus data film.

## Teknologi yang Digunakan
- **Backend**: Node.js & Express.js
- **Database**: MySQL (XAMPP)
- **Tool Testing**: Postman
- **Library**: `mysql2`, `express`
